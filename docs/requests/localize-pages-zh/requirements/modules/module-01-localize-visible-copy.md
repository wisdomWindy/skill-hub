# 模块名称

全站可见文本中文化

# 来源快照

用户要求把当前项目中所有页面的文本改成中文。

# 来源追踪

- 用户当前消息
- 代码扫描范围：`src`、`_data`、`index.html`

# 业务目标

让 SkillHub 页面面向中文用户展示完整、可读、一致的中文文案。

# 用户可见行为

- 页面标题、按钮、输入框占位、空状态、详情区标题、复制反馈、分页按钮等均显示中文。
- 示例技能名称、描述、标签、变更记录显示中文。
- 页面功能行为不变。

# 表单、列表、展示与交互

- 搜索输入：占位提示改中文，输入和路由同步行为不变。
- 排序下拉：选项改为中文。
- 分类按钮：标签改为中文。
- 安装命令复制：按钮和反馈改为中文。
- 空状态：标题、说明、操作按钮改为中文。

# 工作流和状态规则

- 无新增状态。
- 主题切换仍在 dark/light 枚举之间切换，仅显示文案改中文。

# Spec 必须保留的上游约束

- 不更改数据模型和路由。
- 不翻译安装命令、URL、枚举、文件名、代码标识符。

# Page Design 需求

否。

# Architecture Design 需求

否。

# 执行顺序

单模块串行执行。

# 开放问题

无。
