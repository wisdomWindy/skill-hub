# Copyedit Log Template

## Required Artifact Path

- `docs/novel-projects/<novel-id>/artifacts/copyedit-log.md`

## Required Sections

- project identifier
- copyedit pass summary
- style-sheet checks
- naming consistency fixes
- timeline and factual consistency fixes
- diction and tone fixes
- sentence clarity fixes
- unresolved higher-level blockers
- handoff to proofread
